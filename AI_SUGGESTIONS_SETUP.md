# AI Suggestions Feature Setup

This guide explains how to use the new AI-powered suggestions feature in your resume builder.

## What's New

Users can now click an "AI Suggestion" button on various form fields to get AI-generated content recommendations. The system uses Google's Gemini API to provide professional, context-aware suggestions.

## Setup Instructions

### 1. Install Dependencies

**Server side:**
```bash
cd server
npm install
```

The `@google/generative-ai` package has been added to `package.json`.

### 2. Configure Environment Variables

The Gemini API key has been added to `server/.env`:


### 3. Start the Application

```bash
# Terminal 1 - Start server
cd server
npm run dev

# Terminal 2 - Start client
cd client
npm run dev
```

## Features

### AI Suggestion Buttons

The following forms now have AI suggestion buttons:

1. **Work Experience** - Generates professional job descriptions
2. **Projects** - Creates project descriptions based on technologies used
3. **Volunteer Experience** - Writes volunteer work descriptions
4. **Awards & Recognition** - Generates award descriptions
5. **Certifications** - Provides certification context (shows in console)
6. **Publications** - Creates publication descriptions

### How to Use

1. Fill in the basic information (job title, company, technologies, etc.)
2. Click the "AI Suggestion" button below the description field
3. The AI will generate a professional suggestion based on the context
4. The suggestion will automatically populate the description field
5. Edit as needed

## API Endpoint

**POST** `/api/ai/suggest`

Request body:
```json
{
  "fieldType": "jobDescription",
  "context": "Software Engineer at Google"
}
```

Response:
```json
{
  "suggestion": "Professional description generated by AI..."
}
```

## Supported Field Types

- `jobDescription` - Work experience descriptions
- `projectDescription` - Project descriptions
- `volunteerDescription` - Volunteer work descriptions
- `awardDescription` - Award descriptions
- `certificationDescription` - Certification descriptions
- `publicationDescription` - Publication descriptions

## Notes

- The AI suggestion feature requires authentication (JWT token)
- Suggestions are generated in real-time using Gemini API
- Users can edit suggestions before saving
- The feature works best when basic context is provided (job title, company, etc.)
